  • 01コンポジット合成

    実写合成

    セルアニメ合成

    レタッチ編集

    都内老舗CGプロダクションにて新卒から正社員で約5年間修業経験があり、車や飲料のTVCMが得意なプロダクションに在籍していた経験を生かしてフォトリアルな3DCGと実写映像のコンポジット合成が可能です。また劇場版アニメ「ポケットモンスター 水の都の護神 ラティアスとラティオス」のVFX制作とコンポジット合成経験から、セルアニメに光や炎などのCG効果を自然になじませることが得意です。実務経験は15年以上ありますので安心してお任せ下さい。

    使用ツール：AfterEffect / Photoshop

  • 02エフェクト制作

    実写系VFX

    アニメ調VFX

    ゲームエフェクト

    某大手IT企業のゲーム事業部にて、TCGゲームの新規プロジェクトでエフェクトチームの立ち上げからリリースまで約1年間担当しました。 その経験を活かし、プロジェクトの初期設計や、スキル・状態を表現するエフェクト、全画面の必殺技エフェクトの制作が可能です。 また、スマートフォン向けゲームのバトルエフェクト制作にも携わり、直近では「魔法少女まどか☆マギカ外伝 マギアレコード」のバトルエフェクトを担当しました。 2024年以降は某有名オンラインスクールのUnityコースを受講し、趣味でインディーズゲームの制作も行っています。そのため、作成したエフェクトをUnity上で実装することも可能です。 ゲーム開発の流れを理解しているので、スムーズな導入や調整もお任せください。

    使用ツール：Unity / Shyuriken / SparkGear / AfterEffect / Photoshop

  • 03ECサイト制作

    テーマカスタマイズ

    オリジナル納品書

    商品画像作成

    導入するアプリの選定からインストール、初期設定、無料テーマのカスタマイズ、越境サイトの設定まで対応可能です。 カスタマイズ内容としてはメインビジュアルを動画に変更したり、コピーライト表記をオリジナルにすることもできます。 さらに、納品書や領収書のカスタマイズにも対応しており、お店のロゴを追加したり、日本の定率減税に対応する形で異なる税率でも見やすい書類に調整することが可能です。

    使用ツール：Shopify / AdobeXD / Photoshop / Excel

  • 04Webコーディング

    カンプ制作

    既存サイト修正

    WPテーマ制作

    Adobe XDを主軸に "デザインカンプの作成からコーディングまで" 一貫して対応可能です。特にコーディングは、「ライフワークにしたい！」と思うほど大好きで、楽しみながら作業しています。 また、3DCG制作に欠かせないAdobe Photoshopの実務経験を活かし、商品画像の作成や、既存の画像に発生するさまざまな不具合の修正にも対応することが可能です。 まだ実務での経験はありませんが、機会をいただければ丁寧に対応させていただきます。

    使用ツール：Vscode / jQuery / BEM / SCSS / Excel
